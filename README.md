# EyraJs-Template
EyraJs Template


#### Official  template for EyraJs
```js
 import Eyra from 'eyra';
 
 var appModel = new Eyra({
   el: '#app',
   template (){
     return `
       <h1>Hello Eyra</h1>
     `
   }
 })
 appModel.render();
 ```
